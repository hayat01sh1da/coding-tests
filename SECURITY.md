# Security Policy

## Supported Versions

The following table lists which versions of this project currently receive security updates.

| Version              | Supported          |
| -------------------- | ------------------ |
| Latest (main branch) | :white_check_mark: |
| Latest stable release | :white_check_mark: |
| Older releases       | :x:                |
